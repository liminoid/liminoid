<Box variant="danger">
  Python code fences ✨automagically✨ get turned into editors that run in the
  browser!
</Box>

```sh
$ yarn global add liminoid # or `npm install -g liminoid`
$ liminoid new cool-project
$ cd cool-project

$ vsc . # open pwd with VS Code (or your editor of choice)
$ liminoid develop # start live reloading development server

$ liminoid build # compile files into a static standalone bundle

$ yarn global add surge # or `npm install --global surge`
$ surge ./build # deploy to the surge.sh CDN
```

<Box variant="info">

If you want to share your project with the world, it is as easy as uploading
`./build` to a static site host (like [Surge](https://surge.sh/)). See the [Deploying guide](/guides/deployment) for a walkthrough with other hosts.

</Box>
