# Colophon

- Scientific Python [Web Assembly](https://webassembly.org/) modules and loader from [Pyodide](https://github.com/iodide-project/pyodide).
- Components built with [React](https://reactjs.org/) and syntax highlighted by [Prism](https://prismjs.com/).
- Liminoid markdown powered by [MDX](https://mdxjs.com/).
- Tested with [Mocha](https://mochajs.org/) and asserted with [Chai](https://www.chaijs.com/).
- ES6 Javascript transpiled with [Babel](https://babeljs.io/) and bundled using [Rollup](https://rollupjs.org).
- Build tool standing on the shoulders of [webpack](https://webpack.js.org/).
- CLI made pretty with [yargs](http://yargs.js.org/), [chalk](https://github.com/chalk/chalk), [ora](https://github.com/sindresorhus/ora), and [boxen](https://github.com/sindresorhus/boxen).
- Source code formated with [Prettier](https://prettier.io/) and linted with [ESLint](https://eslint.org/).
- Packages hosted on [npm](https://www.npmjs.com/).
- CDN through [jsDelivr](https://www.jsdelivr.com/) and [unpkg](https://unpkg.com/).
- Design remixed from [Rebass](https://rebassjs.org/).
- Site generated with [Gatsby](https://www.gatsbyjs.org/).
- Logo typeset in [Neon 80s](https://indieground.net/product/neon-80s-font/).
